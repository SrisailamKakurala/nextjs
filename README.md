# nextjs
repo for learning nextjs

---

# Cons of ReactJS
- no built-in routing
- no compiler
- no SEO
- no SSR
- no SSG
- no Full-Stack
- no API-Integration
- no Structure

# Pros of NEXTJS
- Hot reloading
- built-in compiler
- SSR & SSG
- SEO Friendly
- Full-Stack
- API Integration
- Built-in Routing
- Performance

---

# Creating a NextJS APP

```bash
npx create-next-app@latest my-app
```
